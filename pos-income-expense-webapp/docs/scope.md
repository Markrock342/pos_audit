# Scope — v1.0.0

## เป้าหมาย

Web App สำหรับร้านกาแฟ บันทึกรายรับ-รายจ่าย ปิดยอดเงินสด ดูสรุป และพิมพ์ใบเสร็จ/เปิดลิ้นชัก

## นอก scope / อนาคต

- Multi-store / multi-org จาก session
- Auth แบบ enterprise (JWT server-side)
- Sync PIN ข้ามเครื่องผ่าน cloud
- Inventory
- Auto-lock หลัง inactive
