# Scope — MVP

## เป้าหมาย

สร้าง Web App สำหรับร้านกาแฟ ใช้บันทึกรายรับ-รายจ่าย ดูสรุปยอด และเตรียมรองรับการพิมพ์ใบเสร็จ / ลิ้นชักเก็บเงินในอนาคต

## ขอบเขต MVP

### 1. บันทึกรายรับ

- เพิ่มรายการรายรับ (ชื่อรายการ, จำนวนเงิน, หมวดหมู่, ช่องทางชำระเงิน, หมายเหตุ)
- แสดงรายการรายรับทั้งหมด
- คำนวณยอดรวมรายรับ

### 2. บันทึกรายจ่าย

- เพิ่มรายการรายจ่าย
- แสดงรายการรายจ่ายทั้งหมด
- คำนวณยอดรวมรายจ่าย

### 3. หมวดหมู่

- สร้าง/แก้ไข/ลบหมวดหมู่ (รายรับ / รายจ่าย)
- กำหนดสีประจำหมวดหมู่

### 4. Dashboard

- สรุปยอดรายรับ-รายจ่ายวันนี้
- สรุปยอดเดือนนี้
- กำไรสุทธิ
- รายการล่าสุด
- กราฟรายรับ-รายจ่าย

### 5. รายงาน

- สรุปรายรับ / รายจ่าย / กำไรสุทธิ ตามช่วงวันที่
- กราฟเปรียบเทียบ
- (อนาคต) ส่งออก PDF / Excel

### 6. พิมพ์ใบเสร็จ

- Template ใบเสร็จสำหรับ preview
- บันทึกเลขที่ใบเสร็จ
- (อนาคต) พิมพ์ผ่าน Thermal Printer

### 7. เตรียมเชื่อม Printer / Cash Drawer

- โครง `/lib/hardware` สำหรับ printer และ cash drawer
- ESC/POS command placeholder
- ยังไม่เชื่อมอุปกรณ์จริงในรอบแรก

## นอกขอบเขตรอบแรก (Out of Scope)

- ระบบ Auth / Role จริง
- เชื่อม Database production
- เชื่อม Thermal Printer / Cash Drawer จริง
- Multi-store / Multi-branch
- Inventory management
